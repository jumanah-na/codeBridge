Assignment ## Instruction

Ensure each student has a computer with a text editor and web browser open. Open a blank document in the text editor to begin coding.

Have students open the saved HTML file in their web browser to view the webpage. Explain how the browser interprets and renders the HTML code to display the webpage.

Instruct students to save their changes in the text editor and refresh the webpage in the browser to see the updates. Encourage students to experiment with different HTML elements and content.
